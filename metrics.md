# Engagement Metrics

Daily Active Users (DAU)

Measures how many users interact with the app daily.

Goal:

Increase weekly engagement by 20%

---

# Retention Metrics

Churn Rate

Measures how many users stop using the app.

Goal:

Keep churn below 5%

---

# Conversion Metrics

Account Creation Rate

Measures how many users create an account after installing the app.

Goal:

Achieve 40% conversion rate

---

# Satisfaction Metrics

User Feedback

Measures user satisfaction and experience quality.

Goal:

Maintain high satisfaction scores
